# 👟 Syntax Wear

Projeto de e-commerce de calçados e moda urbana, desenvolvido com foco em **design moderno**, **organização de código** e **responsividade**.

A proposta do site é oferecer uma experiência visual atrativa para apresentação de produtos, simulando a página inicial de uma loja virtual profissional.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias:

- CSS Grid
- Responsividade
- Git
- GitHub

---

## ✨ Funcionalidades

- Header fixo e moderno
- Menu responsivo para mobile
- Banner principal com chamada de ação
- Seção de categorias de produtos
- Grid visual de produtos em destaque
- Footer com links e redes sociais
- Layout adaptado para diferentes telas
